# 📚 Library Management System

A full-stack Library Management System API built using Node.js, Express, MongoDB, and JWT authentication.
It supports role-based access for Admin and Users, with complete book management, borrowing system, and tracking of due and overdue books.

---

## 🚀 Live Demo

- 🌐 Frontend: https://library-management-system-lyart-nine.vercel.app
- ⚙️ Backend API: https://library-management-system-28ta.onrender.com
